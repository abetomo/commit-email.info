How to use

  1. Find a repository from list
  2. Subscribe to mailing list

How to subscribe

Send an email like the following:

To: ${MAILING_LIST_NAME}@ml.commit-email.info
Cc: null@commit-email.info
Subject: Subscribe
--
subscribe

For example, here is an email to subscribe to mailing list for Ruby's commit email.

To: ruby@ml.commit-email.info
Cc: null@commit-email.info
Subject: Subscribe
--
subscribe

How to unsubscribe

Send an email like the following:

To: ${MAILING_LIST_NAME}@ml.commit-email.info
Subject: Unsubscribe
--
unsubscribe

Body must be empty. You can't use HTML mail because HTML mail has body event when it has empty content.

For example, here is an email to unsubscribe from mailing list for Ruby's commit email.

To: ruby@ml.commit-email.info
Subject: Unsubscribe
--
unsubscribe

Request a new repository

  1. Add a repository and mailing list address to web-hooks-receiver/config.yaml at kou/commit-email.info and send it as a pull request
  2. If you are a owner of the repository, add http://web-hooks-receiver.commit-email.info/ to Webhooks of the repository
  3. If you aren't a owner of the repository, add a repository to github-event-watcher/config.yaml at kou/commit-email.info and send it as a pull request
